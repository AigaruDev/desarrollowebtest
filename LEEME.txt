para instalar el proyecto y funcione correctamente necesitas :
mysql maria-db 10.4 o superior
PHP 8.0 o superior

instalacion: 
el proyecto cuenta con un sql en la cual se encuentra 3 tablas en el archivo "desarrolloweb.sql"
importar en mysql para ser leido las tablas y los registros que posee.

para configuracion de servidor existe un archivo llamado connect.php dentro de la carpeta php 
uri : ./php/connect.php
en el cual puedes agregar las configuraciones de tu servidor o si bien dejar por defecto las 
configuraciones que ya posee que son:

$servidor = "localhost" (ip del servidor o si esta instalado localmente)
$usuario = "root" (nombre de usuario para ingreso a base de datos)
$contrasena = "" (contrase√±a)
$basededatos = "desarrolloweb" (nombre de la base de datos a trabajar)


posterior ya se puede copiar todos los archivos para poder ser visualizados.
